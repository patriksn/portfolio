Overview

This project involved setting up and maintaining a Nightscout server, which is an open-source project that allows continuous glucose monitoring (CGM) data to be stored and accessed from multiple devices. The server runs 24/7 and provides real-time updates of blood sugar levels, making it possible to monitor glucose data from any device with internet access. It should be noted that I did not write any code for this project, but I did learn the very basics of server setup and management (MongoDB database hosted on Heroku).

Key things I learnt

  • How to set up and configure a MongoDB/Heroku server for continuous operation
  • Server maintenance and monitoring to ensure 24/7 availability
  • Working with open-source projects and their community
  • Real-time data synchronization across multiple devices
  • Basic server security and maintenance practices
