Overview

This project focused on comparing the performance of two different ML models for the task to classify human activities (standing, running, walking, walking in stairs) from accelerometer data. The project included data collection, data processing, model training, and model evaluation and comparison. Both models had high performance but the SVM model outperformed the CNN model slightly.

Key Points

  • Collected sensor data from smartphones.
  • Created a data processing pipeline from the raw data to the model input. The data flow can be seen in the image.
  • Feature selection and hyperparameter tuning/model design for the two models.
  • Learnt about evaluation metrics.
  • Gained experience with scikit-learn and TensorFlow.

Technical Skills I learnt

  • Data collection and data pre-processing.
  • Creating a data processing pipeline.
  • Handling uneven distributions in the data.
  • Feature selection and hyperparameter tuning.
  • Model evaluation and comparison.
  • Gained experience with numpy, pandas, scikit-learn and TensorFlow.
  • Learnt the mathematical theory behind SVMs and CNNs (see report for more details).
