Developed an automated apartment hunting assistant that monitors a housing website and notifies users through Discord. A small, simple project made in two afternoons.

Key Features

  • Automated web scraping of a housing website
  • Real-time Discord notifications

Technical Details

The project utilized Python for web scraping and API integration with Discord. It checks a site for new apartment listings and if there are new ones it sends a message with the relevant information in a discord server.
